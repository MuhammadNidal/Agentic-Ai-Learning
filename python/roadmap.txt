1. Python Basics (Foundations)

These are the essentials you must know before doing anything else.

1.1 Introduction to Python

History & Versions – From Python 1.0 to the current version.

Features – Easy syntax, interpreted, dynamically typed, cross-platform.

Installation – Installing Python & pip, environment setup.

Python IDEs – IDLE, VS Code, PyCharm, Jupyter Notebook.

1.2 Syntax & Structure

Indentation – No braces {}, Python uses spaces/tabs.

Comments – Single-line # and multi-line docstrings """...""".

Running Code – Interactive mode (python shell) vs script (.py files).

1.3 Variables & Data Types

Declaring variables (no type needed, Python is dynamic).

Data types:

Numbers: int, float, complex

Strings: str

Boolean: bool

None: NoneType

1.4 Operators

Arithmetic: +, -, *, /, //, %, **

Comparison: ==, !=, <, >, <=, >=

Logical: and, or, not

Assignment: =, +=, -=, etc.

Membership: in, not in

Identity: is, is not

2. Data Structures

Python’s built-in containers for storing data.

2.1 Lists

Creation: my_list = [1, 2, 3]

Indexing & slicing: my_list[0], my_list[1:3]

Methods: append(), remove(), sort(), reverse()

Nested lists & list comprehensions.

2.2 Tuples

Immutable sequences: (1, 2, 3)

Indexing & slicing

Packing & unpacking: a, b = (1, 2)

2.3 Sets

Unique unordered values: {1, 2, 3}

Set operations: union, intersection, difference.

Methods: add(), remove(), discard()

2.4 Dictionaries

Key-value pairs: {"name": "Nidal", "age": 23}

Accessing: dict["name"]

Methods: keys(), values(), items()
